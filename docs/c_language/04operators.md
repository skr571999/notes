# Operators

- Arithmetic Operators

  ```c
  +, -, *, /, %
  ```

- Increment and Decrement

  ```c
  ++a, a++, --a, --a
  ```

  ```c
  int a = 12;
  printf("%d", a++); // 12
  printf("%d", a); // 13
  ```

- Assignment

  ```c
  =, +=, -=, *=, /=, %=
  ```

- Relational

  ```c
  ==, >, <, <=, >=, !=
  ```

- Logical

  ```c
  &&, ||, !
  ```

- Bitwise

|     |                      |
| --- | -------------------- |
| &   | Bitwise AND          |
|     | Bitwise OR           |
| ^   | Bitwise exclusive OR |
| ~   | Bitwise complement   |
| <<  | Shift left           |
| >>  | Shift right          |
|     |                      |

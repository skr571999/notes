# Structures

A structure(struct) is a collection of the variables(can be of the Different types) under a single name.

- Syntax

```cpp
struct structureName{
  dataType member1;
  dataType member2;
}
```

- Example

```cpp
struct Student{
  char name[25];
  int rollno;
  int cityNo;
}
```

- Extensions -- .md, .markdown

```md
**Blod Text**
*Italic Text*
`Inline Code`
~~Strike through~~
>Quoted text

```

- Lists

```md
1. Item 1
2. Item 2

* Item 1
* Item 2

- Item 1
- Item 2
```

- Task List

```md
[x] Task 1
[x] Task 2
```

- Tables

```md
| Column 1| Column 2 |
| ------------ | ------------- |
| Row 1     | Row 1      |
| Row 2     | Row 2      |
```

- Images

```md
![Alert Text](image link)
```

- Code block

